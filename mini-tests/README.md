# Mini Tests – Daily MCQ Practice Drills

This folder contains short MCQ sets designed for **daily practice and concept reinforcement**.

---

## Purpose

Mini tests are used for:

- Quick revision
- Concept reinforcement
- Speed improvement
- Identifying weak topics

---

## Format

Each mini test contains:

- 10 to 25 MCQs
- Single or mixed subject
- Short duration practice

---

## Coverage

- All 10 CS subjects
- Some tests are subject-specific
- Some are mixed-topic drills

---

## Time Limit

- 10–20 minutes per test

---

## How to Use Mini Tests

### STEP 1: Attempt Without Help
- No notes allowed
- No searching

### STEP 2: Immediate Checking
- Verify answers
- Read explanations carefully

### STEP 3: Link to Notes
- Revisit related topic in `/notes`

### STEP 4: Repeat Cycle
- Re-attempt after 1–2 days

---

## Importance

Mini tests are the MOST IMPORTANT part of preparation because they:

- Build speed
- Improve accuracy
- Train MCQ thinking
- Prepare for mock tests

---

## Strategy Tip

If you can consistently score:

- 80%+ in mini tests → You are ready for mock tests
- Below 60% → Revise notes before proceeding
