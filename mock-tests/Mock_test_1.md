### Short Explanations for Selected High-Trap Questions

- **Q8:** Even-parity languages are regular because finite state can track parity; equal-count languages like `a^n b^n` are not regular.  
- **Q15:** Deadlock requires mutual exclusion, hold-and-wait, no preemption, and circular wait; replacing “no preemption” with “preemption” is a classic trap.  
- **Q35:** FIFO can exhibit Belady’s anomaly, while stack algorithms like LRU do not.  
- **Q38:** `{ a^n b^n c^n | n >= 0 }` is a standard example of a language beyond CFL power.  
- **Q58:** NFA may branch, but it is not more expressive than DFA.  
- **Q82:** Object slicing happens when a derived object is copied into a base object by value.  
- **Q100:** With no negative marking, educated guessing after elimination is usually better than leaving blanks.
