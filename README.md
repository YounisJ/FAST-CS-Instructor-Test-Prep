# FAST-NUCES CS Instructor Test Preparation Repository

This repository is a structured and exam-focused preparation system for the Computer-Based MCQ Screening Test for an Instructor position at FAST School of Computing, Karachi Campus.

---

## Objective

To systematically prepare for a medium to advanced level MCQ screening exam (~100 questions) covering core Computer Science subjects required for faculty-level evaluation.

---

## Subjects Covered

- Programming Fundamentals  
- Object-Oriented Programming (OOP)  
- Data Structures & Algorithms (DSA)  
- Databases (DBMS)  
- Operating Systems (OS)  
- Discrete Structures  
- Artificial Intelligence (AI)  
- Theory of Automata (HIGH PRIORITY)  
- Software Engineering  
- Computer Networks  

---



## Repository Structure

/notes        → Subject-wise conceptual revision notes  
/mini-tests   → Daily MCQ practice (10–25 questions)  
/mock-tests   → Full-length mock exams (100 MCQs each) 

---

##  How to Use This Repository

### Step 1: Study Notes
- Go through `/notes` subject by subject  
- Focus on concepts, not memorization  
- Identify MCQ traps and edge cases  
- Give extra attention to Theory of Automata  

---

### Step 2: Mini Tests (Daily Practice)
- Attempt 1–2 mini tests daily  
- Time limit: 10–20 minutes  
- No help or notes allowed  
- Immediately review mistakes  

---

### Step 3: Mock Tests (Full Simulation)
- Attempt full 100 MCQ tests  
- Strict exam conditions  
- No interruptions  
- Analyze results deeply after completion  

---

### Learning Loop

Notes → Mini Tests → Error Fix → Mock Tests → Repeat
